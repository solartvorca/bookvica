## 🚀 Как начать разработку

### Шаг 1: Прочитайте документацию
1. Прочитайте [QUICK_START.md](./QUICK_START.md) (5-10 минут)
2. Посмотрите макеты и выберите вариант реализации
3. Прочитайте [APP_PROMPT.md](./APP_PROMPT.md) для полного понимания

### Шаг 2: Подготовьте окружение
Выберите один из трех вариантов:

**Вариант A: React Web (РЕКОМЕНДУЕТСЯ)**
```bash
npx create-react-app bukvitsa-app
cd bukvitsa-app
npm install tailwindcss zustand
# Следуйте [DEVELOPMENT_GUIDE.md](./DEVELOPMENT_GUIDE.md)
```

**Вариант B: React Native**
```bash
expo init bukvitsa-app
cd bukvitsa-app
npm install react-native-paper
```

**Вариант C: Flutter**
```bash
flutter create bukvitsa_app
# Требует Flutter SDK
```

### Шаг 3: Используйте код и данные
- Скопируйте JSON данные из `bukvitsy_complete.json`
- Используйте примеры компонентов из `DEVELOPMENT_GUIDE.md`
- Следуйте дизайну из `app_config.json`

### Шаг 4: Тестируйте и деплойте
- Протестируйте на мобильных устройствах (симулятор)
- Убедитесь, что работает оффлайн
- Деплойте на Vercel (React Web) или App Store (Native)

## 📊 Содержание 49 буквиц

Приложение включает полную информацию о всех буквицах славянского буквенного строя:

1. **Азъ** (А) - Я, начало, единство
2. **Боги** (Б) - Множественность, движение
3. **Веди** (В) - Знание, видение
... [и еще 46 буквиц]

Каждая буквица содержит:
- Символ (красивое изображение)
- Названием (древнее и современное)
- Смысл (1-2 предложения)
- Семантические модули (4 ключевых аспекта)
- Полное описание
- Эзотерический смысл
- Применение в гадании
- Примеры из культуры
