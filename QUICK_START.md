# 🚀 Быстрый старт - Приложение "Буквица"

## 📋 Что вам нужно знать

Вы готовы создать мобильное приложение "Буквица" - приложение для изучения 49 древних славянских букв с тремя основными режимами: **Учебник**, **Буквица Дня** и **Буквица по дате рождения**.


## 🎯 Три варианта реализации

### Вариант 1: **Web приложение (React + Tailwind)** ⭐ РЕКОМЕНДУЕТСЯ

**Преимущества:**
- Быстрая разработка
- Работает везде (браузер, PWA)
- Простой деплой
- Легко добавлять функции

**Стек:**
```
React 18+ / Next.js
Tailwind CSS
Zustand (state management)
localStorage (data)
PWA (offline)
```

**Шаги:**
```bash
# 1. Создать проект
npx create-react-app bukvitsa-app
cd bukvitsa-app

# 2. Установить зависимости
npm install tailwindcss postcss autoprefixer zustand

# 3. Инициализировать Tailwind
npx tailwindcss init -p

# 4. Скопировать файлы
# - Скопировать bukvitsy_complete.json в src/data/
# - Скопировать компоненты из DEVELOPMENT_GUIDE.md

# 5. Запустить
npm start
```

## 📱 Требования к мобильности

✅ **Обязательно:**
- Работает на iOS и Android
- Оптимизировано для экранов 5"-6"
- Легко читаемый текст (min 14px)
- Крупные кнопки для тача (min 44x44px)
- Быстрые свайпы между буквицами
- Оффлайн работа

✅ **Желательно:**
- Темная тема (по умолчанию)
- Светлая тема (опция)
- Адаптация к разным ориентациям (portrait/landscape)
- Анимации при переключении
- Плавные переходы

## 🔄 Основные экраны и их функции

### 1️⃣ HomeScreen
- Hero секция с приветствием
- Кнопки быстрого доступа
- Буквица дня (карточка)
- История просмотров

### 2️⃣ StudyScreen
- Карточка текущей буквицы (большая)
- Поиск по названию/смыслу
- Фильтр по семантическим модулям
- Навигация (свайп, кнопки, номера)
- Добавление в избранное
- Полная информация по кнопке "Подробнее"

### 3️⃣ DailyScreen
- Случайная буквица дня
- Интерпретация для текущего дня
- История последних буквиц дня
- Возможность получить другую

### 4️⃣ BirthdayScreen
- Выбор даты рождения
- Расчет буквицы личности
- Расчет буквицы судьбы
- Описание характера
- Путь инициации (последовательность развития)

### 5️⃣ FavoritesScreen
- Список избранных буквиц
- История просмотров
- Возможность удалить из избранного
