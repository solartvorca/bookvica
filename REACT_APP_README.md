# 🚀 Приложение "Буквица" - React Web Версия

Полное готовое приложение на **React 18** + **Tailwind CSS** для изучения 49 древних славянских букв.

## 📁 Структура проекта

```
bukvitsa-app/
├── src/
│   ├── components/
│   │   ├── Navigation.tsx         # Нижняя навигация
│   │   ├── BukvitsaCard.tsx       # Карточка буквицы
│   │   ├── SearchBar.tsx          # Поиск
│   │   └── FilterMenu.tsx         # Фильтрация
│   ├── screens/
│   │   ├── HomeScreen.tsx         # Главная страница
│   │   ├── StudyScreen.tsx        # Учебник (все 49 буквиц)
│   │   ├── DailyScreen.tsx        # Буквица дня
│   │   ├── BirthdayScreen.tsx     # По дате рождения
│   │   └── FavoritesScreen.tsx    # Избранное
│   ├── store/
│   │   └── bukvitsyStore.ts       # Zustand store (состояние)
│   ├── utils/
│   │   └── birthdayCalculator.ts  # Утилиты для расчетов
│   ├── data/
│   │   └── bukvitsy.json          # База 49 буквиц
│   ├── types.ts                   # TypeScript типы
│   ├── App.tsx                    # Главный компонент
│   ├── main.tsx                   # Входная точка
│   └── index.css                  # Глобальные стили
├── index.html                     # HTML файл
├── package.json                   # Зависимости
├── vite.config.ts                 # Конфиг Vite
├── tailwind.config.js             # Конфиг Tailwind
├── postcss.config.js              # Конфиг PostCSS
└── tsconfig.json                  # Конфиг TypeScript
```

### Локальное хранилище
Приложение использует `localStorage` для сохранения:
```javascript
bukvitsa_favorites      // ID избранных буквиц
bukvitsa_history        // ID просмотренных буквиц
bukvitsa_daily_history  // История буквиц дня
user_birthday           // Дата рождения пользователя
```

### Состояние приложения (Zustand)
```typescript
useBukvitsyStore() // Главное хранилище
├── bukvitsy        // Все 49 буквиц
├── favorites       // Избранные (ID)
├── history         // История просмотров (ID)
├── dailyHistory    // История буквиц дня
└── методы для работы с данными
```

## 📝 Особенности реализации

### Поиск и фильтрация
- Поиск по названию, смыслу и транслитерации
- Фильтрация по одному или нескольким семантическим модулям
- Реактивное обновление результатов

### История просмотров
- Ограничена 20 записями
- Сохраняется автоматически при просмотре
- Выводится в отдельной вкладке

### Расчет буквицы по дате
```javascript
personalityNumber = sum(digits(day))
destinyNumber = sum(digits(day + month))
characteristicNumber = (personalityNumber + destinyNumber - 2) % 9 + 1
```

### Бизнес-логика
- Буквица дня выбирается один раз в день
- При повторном входе в день показывается та же буквица
- История обновляется каждый день
