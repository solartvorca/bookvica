# Промпт для создания приложения "Буквица - Древняя Славянская Мудрость"

## 📱 概览 приложения

Создать полнофункциональное мобильное приложение для изучения 49 древних славянских буквиц (дохристианский буквенный строй). Приложение должно быть интерактивным, красивым, образовательным и мистическим одновременно.

## 🎯 Основные режимы

### 1️⃣ **РЕЖИМ "УЧЕБНИК" (Study Mode)**
- Полная база данных 49 буквиц
- Для каждой буквицы отображать:
  - **Визуализация**: Красиво оформленная буквица (как на изображении - красный символ на золотом фоне)
  - **Название и транслитерация**: Исконное славянское имя + латинская транскрипция
  - **Буквица (символ)**: Древний славянский символ
  - **Основное значение**: Краткая суть буквицы (1-2 предложения)
  - **Семантические модули**: 4 ключевых аспекта/смысла буквицы
  - **Развёрнутое описание**: Полная история, эзотерический смысл, применение в магии/гадании
  - **Примеры и истории**: Иллюстрирующие примеры из славянской культуры

- **Навигация в учебнике**:
  - Свайп влево/вправо между буквицами (как в карточках)
  - Быстрый переход по номерам (меню с выбором)
  - Поиск по названию
  - Фильтр по семантическим модулям

### 2️⃣ **РЕЖИМ "БУКВИЦА ДНЯ" (Daily Рune)**
- Каждый день случайная или циклическая буквица "дня"
- Отображает:
  - Красивую карточку с буквицей дня
  - Её значение и семантические модули
  - Интерпретацию для текущего дня (совет на день)
  - Дневной совет (рекомендация на основе буквицы)
  - История дня (как эта буквица может помочь сегодня)

- **Функционал**:
  - Ежедневное обновление (можно пропустить, выбрать другую)
  - Сохранение истории буквиц дня (просмотр прошлых дней)
  - Уведомления (опционально) каждое утро

### 3️⃣ **РЕЖИМ "БУКВИЦА ПО ДАТЕ РОЖДЕНИЯ" (Birthday Рune)**
- По дате рождения пользователя вычислить личную буквицу
- **Алгоритм расчета**:
  - День месяца: первая цифра дня + первая цифра месяца (итоговое число от 1-49)
  - ИЛИ полный расчет нумерологии (сумма всех цифр даты)
  - ИЛИ простой: день + месяц, если больше 49, вычесть 49
  
- **Отображение результата**:
  - Буквица Личности (основная энергия)
  - Буквица Судьбы (жизненное предназначение)
  - Описание характера, талантов, вызовов
  - Совместимость с другими буквицами
  - Рекомендации по развитию
  - Путь инициации (последовательность смежных буквиц)

## 🎨 Дизайн и UX

### Мобильный интерфейс
- **Отступы и размеры**: Оптимизировано для смартфонов (320px - 480px - 600px ширина)
- **Шрифты**:
  - Основной текст: читаемый без засечек (Roboto, Inter, Open Sans)
  - Названия буквиц: выразительный шрифт (GadugiBook, Palatino, Cinzel для эффекта древности)
  - Символы буквиц: крупные (min 72px), контрастные
  
- **Палитра цветов**:
  - Основной фон: тёмный (темно-синий #1a2a3a или черный #0a0a0a) - ночная магия
  - Карточки: тёплые (золотой #d4a574, кремовый #f5e6d3, красный #c41e3a для символов)
  - Акценты: красный (#c41e3a), золотой (#d4a574), серебристый (#e8e8e8)
  - Текст: белый/кремовый на темном фоне для контраста
  
- **Элементы дизайна**:
  - Символы буквиц отображаются как на изображении (красный символ на золотом квадратном фоне)
  - Текстуры: древний пергамент, кожу, лён (тонкие текстуры фонов)
  - Анимации: плавные переходы при смайпе, легкие свечения вокруг символов
  - Орнаменты: славянские узоры по краям карточек (опционально)
